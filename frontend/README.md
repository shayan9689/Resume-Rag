# Arabic Docs RAG Frontend

React.js frontend for the Arabic Docs RAG project.

## Install

```bash
cd frontend
npm install
```

## Run

```bash
npm start
```

Opens at `http://localhost:3000` (or next free port).

## Build

```bash
npm run build
```

## Config

API URL is in `src/App.js` (default `http://localhost:8000`).
